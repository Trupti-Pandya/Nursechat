## Medical Screening Assistant - A Chatbot to Help Nurses

**By Trupti Pandya**

### Project description:

This project explores how a chatbot can assist nurses in the initial patient screening process. In order to obtain basic health information, the chatbot will engage with patients by posing structured questions. It will recommend which hospital ward or department could be best suited for additional evaluation based on the responses. The process will always be supervised by nurses, who have the authority to change or disregard suggestions as necessary.

The purpose of the system is to assist nurses, not to take their position. It will help nurses save time and concentrate on more important patients by cutting down on repetitious questions. To improve the ability of Chatbot to provide relevant responses, AI techniques like RAG  will be explored to allow chatbot to answer hospital-specific queries.

This project will be built step by step, focusing on practical improvements and ensuring that the system remains easy to use for both patients and nurses. The goal is to make a system which will be helpful for nurses.
